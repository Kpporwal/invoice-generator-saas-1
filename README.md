# 📌 About

Invoice Generator SaaS is a professional GST Invoice Management System designed for businesses, freelancers and agencies.

It helps users generate professional invoices with GST calculations, download PDFs, print invoices and securely manage invoice history.

# 🚀 Installation

Clone repository

```bash
git clone https://github.com/Kpporwal/invoice-generator-saas-1.git
```

Install packages

```bash
npm install
```

Run

```bash
npm run dev
```

Build

```bash
npm run build
```
