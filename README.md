# Invoice Generator SaaS

A modern GST Invoice Generator built with React, TypeScript, Tailwind CSS and Supabase.

## Features

- Create GST Invoices
- Edit & Delete Invoices
- Invoice History
- PDF Download
- Print Invoice
- Company Logo
- Digital Signature
- QR Code Support
- GST Calculation
- Responsive Design
- Secure Authentication

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- Supabase
